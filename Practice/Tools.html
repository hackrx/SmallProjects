BUNDLE PHOBIA

This is a web application that allows you to determine how much npm package will add to your total bundle size. If you are working with a modern front-end framework, you want to try and keep the bundle size as small as possible to optimize the initial page load to new users.Bundle Phobia makes it easy to analyze the addition of a new dependency to your project.What you need to do is just enter the package name and it will tell you the bundle size , the number of download times, weather or not it is a tree shaker bowl as well as a bunch of other information for every single version of that package and you can even upload your entire package JSON to see all of your dependencies ranked by size.
